Place of classification in machine learning.
	Машинное обучение на высоком уровне бывает контролируемым (supervised) и неконтролируемым (unsupervised).

	При supervised:
		- У нас есть ярлык или вещь, которую мы пытаемя предсказать.
		- Мы помечаем исторические данные (данные, прошедших тестов), которые будем использовать для корректировки нашей модели.
		- Используется 2 термина: КЛАССИФИКАЦИЯ и РЕГРЕССИЯ.

	Целью классификации является категориальное значение (часто True or False, но может быть несколько категорий).
		Пример:
		Прогнозирование того, что кто-то не выполнит свои обязательства по кредиту. (Классификация)

	Целью регресии является получение числового значения.
		Пример:
		Прогнозирование цен на жильё. (Регрессия)
	
	Логистическая регрессия прелставляет собой алгоритм решения проблем классификации.

	При unsupervised:
		- Нет конкретной цели, которую надо достигнуть.

	Терминология классификации (Classification Terminology):
		Пример из Titanic dataset:
		- Столбец "Survived" - это того, что мы пытаемся предугадать. Мы называем это целью (target).
		- Все последующие столбцы - это информация о каждом пассажире, котораю мы будем использовать для предсказания цели. Мы назваем эту каждый из этих столбцов свойством (feature).
		- Machine Learning MODEL - модель, основанная на уже изученных компьютером данных, позволяющая предсказать результат для другого объекта, с ещё не встречавшимися свойствами.

	P.S. Иногда свойства могут называтся предсказателями (predictors)
