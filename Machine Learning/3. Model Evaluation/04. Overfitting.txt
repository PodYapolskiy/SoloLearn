Overfitting - Переобучение.
	Training Set и Test Set (Обучающий набор и тестовый набор)
	
	Наша модель будет делать прогноз на основе данных, на которые мы не знаем ответа. Из-за этого нужно оценить, насколько хорошо наша модель работает с новыми данными, а не только с уже просмотренными данными.
	Чтобы смоделировать прогнозирование новых 'невидимых' (возможных) данных, мы должны	разбить наш набор данных на 2:
		- Training Set. Обучающий набор исп. для построения моделей.
		- Test Set. Тестовый набор исп. для оценки моделей.

 	Мы разделяем наши данные перед построением модели, поэтому модель не знает набора тестов, и мы дадим ей справедливую оценку.

 	Для примера возьмём набор данных из 200 экземпляров данных:
	 
		Training Set(150 datapoints) | Test Set(50 datapoints)
				Build model					Evaluate model

	Стандартная разбивка: 70-80% данных в Training Set, а оставшиеся 20-30% в Test Set.

	Плюсы и минусы перевеса в разные стороны:
		Использование меньшего кол-ва данных в Training Set означает, что наша модель не будет иметь стольких данных, на которых можно было бы учиться. Поэтому надо предоставить ей как можно больше, оставив при этом достаточно данных для Test Set.

	Если мы будем использовать слишком много данных для обучения, у нас будет недостаточно данных для тестов, и тесты могут показать неточный результат.

	Если мы будем использовать слишком много данных для тестов, у нас будет недостаточно данных для обучения, и модель получиться очень расплывчатой, зато результаты тестов будут максимально точными.
