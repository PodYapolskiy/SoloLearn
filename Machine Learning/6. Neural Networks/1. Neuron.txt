Нейрон (Neuron):
    Нейрон или искуственный нейрон (Artificial neuron) - объект, вдохновлённый человеческими нейронами, который умеет принимать ввод, выполнять вычисления и выдавать результат.

    Пример (1.jpg):
        Нейрон принимает ввод: x1, x2, выполняет вычисления и выдаёт вывод y1

    Нейроны не ограничены в количестве входных данных и выходных.


Вычисления нейрона (Neuron Computations):
    1) При вычислениях в нейроне мы сначала помещаем входные данные в следующее выражение:

        w1 * x1 + w2 * x2 + b
    
    где "w1" и "w2" - веса (weights), а "b" - смещение (bias).

    2) Далее, мы вставляем это значение в функцию активации (activation function). Функция активации сжимает полученное значение к промежутку от 0 до 1.

    Чаще всего функцией активации является сигмоида (sigmoid):

                         1
        sigmoid(x) = ----------
                      1 + e^-x
    
                                                  1
    y = f(w1 * x1 + w2 * x2 + b) = --------------------------------
                                    1 + e^-(w1 * x1 + w2 * x2 + b)

    Функция активации - f(x) (в данном случае сигмоида),
    w1, w2, b - контролируют работу нейрона и называются параметрами,
    y - выходное значение нейрона.


Функции активации:
    1) Рассмотренная выше сигмоида
    2) tanh

                          sinh(x)     e^x - e^-x
        f(x) = tanh(x) = --------- = ------------
                          cosh(x)     e^x + e^-x
    
    Похожа на сигмоиду, но возвращает значение из диапазона [-1; 1] 

    3) ReLU

                    0 if x <= 0
        ReLU(x) =       and
                    x if x > 0

    Выпрямленная линейная единица (Rectified Linear Unit).
    Функция идентификации для положительных чисел, которая образует отрицательные значения в 0.


Пример:
    Имеем нейрон, принимающий 2 значения, выводящий 1 значение и функцию активации сигмоиду.

    Weights (w1, w2) = [0, 1]
    Bias (b) = 2

    Если даём на вход (1, 2):

        w1 * x1 + w2 * x2 + b = 0 * 1 + 1 * 2 + 2 = 4
        y = f(w1 * x1 + w2 * x2 + b) = 1 / (1 + e^-4) = 0.9820
    
    Если даём на вход (2, -2):

        w1 * x1 + w2 * x2 + b = 0 * 2 + 1 * (-2) + 2 = 0
        y = f(w1 * x1 + w2 * x2 + b) = 1 / (1 + e^0) = 0.5
